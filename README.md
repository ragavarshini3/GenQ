# GenQ – AI Question Paper Generator

GenQ is a Flask-based AI-powered web application that generates question papers from past papers using Google Gemini AI. This project helps students and teachers automatically generate exam-ready question papers quickly and efficiently.

---

## Features

- User Registration and Login system
- AI-powered question paper generation using Gemini AI
- Flask backend integration
- Stores users and past papers in JSON format
- Clean and simple user interface
- Fast and lightweight system
- Easy to install and run

---

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python
- Flask

AI Integration:
- Google Gemini API

Database:
- JSON files (users.json, past_papers.json)
